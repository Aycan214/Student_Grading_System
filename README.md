# Student Grading System (Java & MySQL) 🎓

Bu proje, NetBeans IDE kullanılarak geliştirilmiş, öğrenci notlarını ve devamsızlıklarını takip eden masaüstü tabanlı bir yönetim sistemidir. 

---

### 🇹🇷 Proje Özellikleri (Turkish)
Bu uygulama, eğitim kurumları için temel öğrenci verilerini yönetmek amacıyla tasarlanmıştır.

* **Modüler Yapı:** Ana menü, Not Giriş Sistemi ve Devamsızlık Takibi gibi 5 ana modülden oluşur.
* **Veritabanı Yönetimi:** Kayıtların dinamik olarak tutulması için **MySQL** entegrasyonu kullanılmıştır.
* **İlişkisel Mimari:** Veritabanı tablosu tasarımında **Primary Key (PK)** ve **Foreign Key (FK)** yapıları kullanılarak veri tutarlılığı en üst düzeye çıkarılmıştır.
* **CRUD İşlemleri:** Öğrenci ekleme, silme, güncelleme ve listeleme özellikleri tam işler haldedir.

### 🇬🇧 Project Features (English)
A desktop-based management system developed in NetBeans for tracking student grades and attendance.

* **Modular Design:** Includes 5 core modules: Main Menu, Grade Entry, and Attendance Tracking.
* **Database Integration:** Powered by **MySQL** for robust and persistent data management.
* **Relational Architecture:** Implements **Primary Key (PK)** and **Foreign Key (FK)** constraints to ensure data integrity.
* **Full CRUD Support:** Functional features for adding, deleting, updating, and listing student records.

---

### 🛠️ Teknik Detaylar / Technical Stack
* **Language:** Java (JDK)
* **IDE:** NetBeans
* **Database:** MySQL
* **GUI Library:** Java Swing / AWT
